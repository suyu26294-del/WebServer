⚡ CppWebServer

基于 C++17 + epoll + Reactor 的高性能 HTTP 服务器

🔄 Reactor 事件驱动

单 Reactor + 线程池模型。主线程负责 epoll 事件分发，工作线程执行 HTTP 解析与响应构建，最大化并发吞吐。

⚡ ET 非阻塞 I/O

连接 fd 采用 EPOLLET + EPOLLONESHOT 模式，配合非阻塞 Socket 循环读写，减少系统调用次数。

🔧 线程池

固定大小线程池，任务队列以条件变量驱动，避免线程频繁创建销毁带来的开销。

⏱ 定时器

基于小根堆的定时器管理器，O(log n) 超时注册与调整，支持 keep-alive 连接超时回收。

📝 异步日志

前台线程写入有界阻塞队列，后台日志线程批量落盘，按天切分，四级日志（DEBUG/INFO/WARN/ERROR）。

🗄 数据库连接池

预建 MySQL 连接池，RAII 守卫防止连接泄漏，预处理语句防止 SQL 注入。

🗺 HTTP 状态机

有限状态机解析 HTTP/1.1 请求，正确处理半包/粘包，支持 GET/POST、URL 解码、keep-alive。

📦 mmap 零拷贝

静态文件通过 mmap + writev 聚合发送，减少用户态/内核态数据拷贝，提升静态资源吞吐。

⚙ 配置化管理

INI 风格配置文件管理端口、线程数、超时、日志级别、数据库参数，无需重新编译即可调整。
